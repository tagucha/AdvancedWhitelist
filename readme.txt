ホワイトリストを少し拡張するプラグインです。
配信者さんが企画をやっている際に「一度落ちてしまった人がホワイトリストの影響で再び鯖に入ることができない」
などの状況をなくしたくて作りました。(自分が落ちた時に悲しかった)
ver1.14.4の使用を前提としていますが、他のバージョンでも動くと思います。
1.14.4のみ動作確認をしています。

コマンド
/whitelist : 接続が許可されているアカウントを表示します。
- on : ホワイトリストを有効にします。また、オンラインのプレイヤーは一時的なホワイトリストへ追加されます。
- off : ホワイトリストを無効にします。また、一時的なホワイトリストに追加されていたプレイヤーは削除されます。
- add [instant|eternal] <player> : ホワイトリストにプレイヤーを追加します。instant:一時的,eternal:永続
- remove <player> : ホワイトリストからプレイヤーを削除します。
- reload : whitelist.ymlを再読み込みします。
- clear : 一時的なホワイトリストに追加されているプレイヤーを全て削除します。
- help : コマンドの説明を表示します。

説明
このプラグインを導入するとホワイトリストが二種類に分かれます。
永続するものと一時的なものです。
永続するものはデフォルトのホワイトリストと同様の効果です。
一時的なものはホワイトリストが有効になった際にオンラインのプレイヤーが追加され、無効になると同時に全て削除されます。
UUIDで管理しているのでプレイヤーの名前変更による影響はありません。

権限
whitelist.command : このプラグインのコマンドを使うための権限です。
whitelist.join : ホワイトリストに登録されていなくても鯖に入るための権限です。
